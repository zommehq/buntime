== Referência de Comandos

=== Plugin Commands

==== buntime plugin

Lista todos os plugins instalados no servidor Buntime.

*Sintaxe:*

[source,bash]
----
buntime plugin [options]
----

*Opções:*

[cols="1,1,2"]
|===
| Opção | Default | Descrição

| `--url <url>`
| `http://localhost:8000`
| URL do servidor Buntime

| `-k, --insecure`
| `false`
| Ignora verificação de certificado TLS
|===

*Exemplo:*

[source,bash]
----
# Listar plugins no servidor local
buntime plugin

# Listar plugins em servidor remoto
buntime plugin --url https://buntime.home

# Listar plugins com certificado self-signed
buntime plugin --url https://buntime.home -k
----

*Saída:*

[source]
----
Installed Plugins (3)
Name                          Base       Versions
----------------------------  ---------  --------
@buntime/plugin-keyval        /keyval    1.0.0
@buntime/plugin-deployments   /deploy    1.0.0
@buntime/plugin-metrics       /metrics   1.0.0
----

==== buntime plugin install

Instala um plugin a partir de um arquivo tarball (.tgz).

*Sintaxe:*

[source,bash]
----
buntime plugin install <file> [options]
----

*Argumentos:*

[cols="1,3"]
|===
| Argumento | Descrição

| `<file>`
| Caminho para o arquivo tarball (.tgz ou .tar.gz)
|===

*Opções:*

[cols="1,1,2"]
|===
| Opção | Default | Descrição

| `--url <url>`
| `http://localhost:8000`
| URL do servidor Buntime

| `--token <token>`
| -
| Token de autenticação para upload

| `-k, --insecure`
| `false`
| Ignora verificação de certificado TLS
|===

*Exemplo:*

[source,bash]
----
# Instalar plugin local
buntime plugin install ./my-plugin.tgz

# Instalar em servidor remoto com autenticação
buntime plugin install ./my-plugin.tgz \
  --url https://buntime.home \
  --token eyJhbGciOiJIUzI1NiIs...

# Instalar com certificado self-signed
buntime plugin install ./my-plugin.tgz \
  --url https://buntime.home -k
----

*Saída:*

[source]
----
Installing plugin: ./my-plugin.tgz
⠋ Uploading and extracting...

Plugin installed successfully!
  Name: @company/my-plugin
  Version: 1.0.0
  Path: /data/plugins/@company/my-plugin/1.0.0
----

==== buntime plugin remove

Remove um plugin instalado. Interface interativa para seleção.

*Sintaxe:*

[source,bash]
----
buntime plugin remove [options]
----

*Opções:*

[cols="1,1,2"]
|===
| Opção | Default | Descrição

| `--url <url>`
| `http://localhost:8000`
| URL do servidor Buntime

| `--token <token>`
| -
| Token de autenticação para remoção

| `--force`
| `false`
| Pula confirmação interativa

| `-k, --insecure`
| `false`
| Ignora verificação de certificado TLS
|===

*Exemplo:*

[source,bash]
----
# Remover plugin interativamente
buntime plugin remove

# Remover sem confirmação
buntime plugin remove --force

# Remover de servidor com certificado self-signed
buntime plugin remove --url https://buntime.home -k
----

*Fluxo Interativo:*

[source]
----
Select a plugin to remove:
❯ @buntime/plugin-keyval (2 versions)
  @buntime/plugin-metrics (1 version)

Select version to remove for @buntime/plugin-keyval:
❯ All versions
  1.0.0
  1.1.0

Are you sure you want to remove @buntime/plugin-keyval (all versions)?
Press Y to confirm, N or Escape to cancel

Plugin removed: @buntime/plugin-keyval (all versions)
----
