# Buntime Wiki

Canonical knowledge base for the [Buntime](https://github.com/zommehq/buntime) project — a Bun runtime with isolated worker pool, plugin system, and micro-frontend shell.

Start here:

1. [`index.md`](./index.md) — navigable catalog (by audience, topic, source).
2. [`CONVENTIONS.md`](./CONVENTIONS.md) — schema, frontmatter, named operations.
3. [`log.md`](./log.md) — change log (most recent first).
4. [`QMD.md`](./QMD.md) — local search setup (named index `buntime`).

Agent execution rules and the wiki-vs-repo policy live in the repo root [`/CLAUDE.md`](../CLAUDE.md) (`/AGENTS.md` is a symlink to it).
